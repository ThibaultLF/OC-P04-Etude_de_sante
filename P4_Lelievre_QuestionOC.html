Remarque 1: Relation avec la disponibilité intérieure
¶

Par intuition, nous supposons que la disponibilit intrieure est gale la faon dont nous obtenons le produit et galement la faon dont nous le consommons.

Voici les formules: $$Dispo\ intrieure = Production + Importations Exportation + Variation\ de\ stock$$ $$Dispo\ intrieure = Nourriture + Pertes + Aliments\ pour\ animaux + Autres\ utilisations + Semences + Traitement$$

     vector = vector/sum(vector)*2*pi
     temp = c()
     j = 0
     l = 0
     for (i in 1:length(vector)) {
          k = vector[i]/2
